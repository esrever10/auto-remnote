# auto-remnote
Some autohotkey scripts for Remnote

## V1.8.7 特性
---

- 自动识别 Remnote Windows 客户端和主流浏览器进程，不影响其他应用
+ 连续中文字符自动转Remnote对应英文字符命令
    - 输入 》》 自动转 >> 
    - 输入 《《 自动转 << 
    - 输入 ：：自动转 :: 
    - 输入 ；；自动转 ;;
    - 输入 ！！自动转 !!
    - 输入 ## 自动转 ##
    - 输入 %% 自动转 %%
    - 输入 …… 自动转 ^
    - 输入 —— 自动转 _
